KolamWeaver

A Technical White Paper on Weaving Tradition into the Digital Age

Abstract

Kolam (also known as Muggu or Rangoli) is a traditional Indian art form rich with mathematical ingenuity, including principles of symmetry, recursion, and graph theory. This document outlines **KolamWeaver**, an intelligent platform designed to bridge this ancient knowledge with modern technology. The system uses a computer vision and algorithmic approach to analyze images of Kolams, decode their fundamental design principles, and recreate them digitally. Furthermore, it empowers users to learn, create, and experience these designs in an interactive web environment and through immersive Augmented Reality (AR), making this cultural heritage accessible, understandable, and innovative for a new generation.

Problem Statement

This project addresses the challenge from AICTE's Indian Knowledge Systems (IKS) division: "Develop computer programs to identify the design principles behind the Kolam designs and recreate the kolams." The core task is to move beyond simple image replication and develop a system that can computationally understand and formalize the artistic and mathematical rules embedded in these traditional patterns.

Proposed Solution: The KolamWeaver Ecosystem

KolamWeaver is a comprehensive web platform that provides a complete cycle of analysis, creation, and experience. It functions as an intelligent ecosystem that decodes tradition and makes it a canvas for future creativity.

Key Features:

  • Analyze & Decode: Users can upload an image of any Kolam, and our system's analysis engine will extract its core principles—dot grid, symmetry, path logic—presenting its mathematical 'DNA'.
  • Create & Learn: An interactive web canvas allows users to design their own Kolams from scratch, assisted by innovative tools like a "Symmetry Brush" that make complex geometric concepts intuitive.
  • Experience in AR: Users can project their digital Kolam creations into their real-world environment using our web-based Augmented Reality feature, visualizing the art in its intended context.

Technical Architecture

Our solution is built on a modern, robust, and open-source tech stack, architected for efficiency and scalability. The system is divided into a backend analysis engine and a frontend user experience.

System Architecture Diagram

Technology Stack:

  • Backend: Python, Flask, OpenCV, scikit-image, NetworkX.
  • Frontend: HTML5, CSS3, JavaScript (ES6+), Three.js.
  • Augmented Reality: WebXR Device API.

Our Rationale: Algorithmic Precision over AI Prediction

We consciously chose a direct algorithmic method over a traditional machine learning model. The goal is to extract explicit, understandable rules, not just classify a pattern. An algorithmic approach is data-efficient, working on a single image without needing a massive training dataset, and it provides the precise, explainable output required to truly understand and recreate the art form's logic.

Impact and Benefits

Beyond the technical implementation, KolamWeaver is designed to create a lasting positive impact.

  • Cultural Preservation: We are creating a living, digital archive of not just the Kolam images, but their underlying mathematical soul.
  • STEAM Education: Our platform makes complex geometry and computational thinking feel like a creative game, inspiring a new generation of learners.
  • Global Accessibility: We offer a modern, immersive way for a global audience to experience and interact with a profound piece of Indian heritage.
  • Creative Empowerment: The platform serves as a novel tool for artists and designers to innovate on traditional patterns for modern applications.

Project Roadmap & Next Steps

Following the successful submission of this white paper, the development of the KolamWeaver prototype will commence. The work is divided into a clear, phased to-do list.

  • Setup Python backend with Flask and define API endpoints.
  • Implement the core OpenCV image preprocessing pipeline (grayscale, binarization, noise reduction).
  • Develop and fine-tune the dot detection algorithm using Hough Circles.
  • Implement line skeletonization and build the graph data structure using NetworkX.
  • Build the first generative algorithm for simple, symmetrical patterns from a rule set.
  • Create the basic HTML/CSS frontend structure and user interface.
  • Develop the interactive "Creator" tool using the HTML Canvas API and JavaScript.
  • Connect the frontend to the backend API for a full analysis-to-display loop.
  • Integrate Three.js and the basic WebXR scene for the Augmented Reality feature.
  • Deploy the initial working prototype to a cloud service for live demonstration.
